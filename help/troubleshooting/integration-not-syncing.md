# Integration not syncing

Troubleshooting steps when your integration is not syncing data.

## Quick checks

1. **Verify integration status**
   - Check if integration is active
   - Look for error indicators
   - Review recent run history

2. **Check connection**
   - Test API credentials
   - Verify network connectivity
   - Check system availability

3. **Review configuration**
   - Verify data mappings
   - Check filter settings
   - Review scheduling rules

## Common issues

- **Inactive integration**: Integration may have been paused
- **Connection errors**: API credentials may be invalid
- **Data filters**: Filters may be too restrictive
- **Scheduling issues**: Sync schedule may be misconfigured

## Solutions

- Restart the integration
- Update credentials
- Adjust filters
- Review and fix scheduling
