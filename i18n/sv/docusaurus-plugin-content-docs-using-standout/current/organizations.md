## Redigera en organisation

**Obs**: Endast användare med Adminroll kan redigera organisationsdetaljer.

1. Gå till Organisationssidan
2. Klicka på organisationsnamnet
3. Klicka på "Redigera"
4. Uppdatera de fält du vill ändra
5. Klicka på "Spara" för att tillämpa ändringarna

## Planalternativ

Organisationer har olika planalternativ:

- **Starter**: Grundplan med begränsade funktioner
- **Pro**: Fler funktioner och högre gränser
- **Premium**: Alla funktioner och högsta gränser
- **Anpassad**: Skräddarsydd plan för specifika behov

Vissa organisationer kan ändra sin plan, andra kan inte. Detta beror på organisationsinställningarna och ditt avtal med oss.

Om ditt konto skapades av en partner eller agent kan du ha andra planalternativ tillgängliga. Ditt konto kan också täckas av partnern och då kanske du inte har tillgång till fakturerings- eller planalternativ från oss. Kontakta din partner eller agent för att få mer information om dina alternativ.

## Hantera underorganisationer

Vissa planer tillåter att skapa underorganisationer:

- Underorganisationer ärver vissa inställningar från föräldern
- De kan ha sina egna integrationer och anslutningar
- Anslutningar kan delas från föräldraorganisationen till underorganisationen.
- Medlemmar kan bjudas in till underorganisationen utan möjlighet att komma åt föräldraorganisationen.
- Använd detta för att organisera stora organisationer med flera divisioner eller avdelningar.
