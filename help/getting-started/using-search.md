The search feature helps you quickly find organizations and integrations.

## Opening search

Press the **"s"** key anywhere in the application to open the search field.

You can also click on the search icon in the top navigation bar.

## What you can search for

The search feature finds:

- **Organizations**: Search by organization name
- **Integrations**: Search by integration name

## How to search

1. Press **"s"** or click the search icon
2. Type your search term
3. Results appear as you type
4. Click on a result to open it

## Tips

- Search works in real-time as you type
- Use clear keywords for better results
- The search finds partial matches in names
