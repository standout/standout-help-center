Anslutna konton (kallas också anslutningar) är de tjänster du har länkat till Standout, som exempelvis Shopify, Fortnox, Stripe. Varje anslutning låter dig reagera på händelser eller utföra åtgärder på den tjänsten.

## Visa konton i Kontosektionen

Gå till sektionen "Konton" i en organisation för att se alla anslutna konton.

- Varje konto visar sitt namn och typ
- Listan visar konton för den aktuella organisationen
- Klicka på ett konto för att se dess detaljer

**Obs**: Endast användare med Admin- eller Medarbetarroll kan visa kontodetaljer. Användare med Läsarroll kan inte öppna konton för att se deras detaljer.

## Lägga till ett nytt konto

**Obs**: Endast användare med Admin- eller Medarbetarroll kan lägga till nya konton.

1. Gå till Kontosidan i en organisation
2. Klicka på "Nytt konto"
3. Välj kontotypen (tjänst du vill ansluta)
4. Följ anslutningsstegen:
   - Ange nödvändiga inloggningsuppgifter
   - Auktorisera anslutningen
   - Bekräfta anslutningen
5. Kontot visas i din lista

## Kontodetaljer

Om du har Admin- eller Medarbetarroll kan du öppna ett konto för att se dess detaljer. När du öppnar ett konto kan du se:

- **Namn**: Kontonamnet
- **Typ**: Vilken tjänst det ansluter till
- **Status**: Om det är anslutet och fungerar
- **Anslutningsinfo**: Detaljer om anslutningen
- **Metadata**: Ytterligare information om kontot
- **Integrationer**: Vilka integrationer som använder detta konto
- **Fel**: Eventuella anslutnings- eller användningsfel

## Hantera kontoinställningar

**Obs**: Endast användare med Admin- eller Medarbetarroll kan redigera kontoinställningar. Alla användare kan visa anslutningsstatus, integrationer och fel om de kan se kontodetaljerna.

Du kan:

- **Redigera**: Uppdatera kontonamn eller inställningar (endast Admin och Medarbetare)
- **Visa anslutning**: Kontrollera anslutningsstatus
- **Visa integrationer**: Se vilka integrationer som använder detta konto
- **Visa fel**: Kontrollera eventuella problem

## Kontofel

Om ett konto har fel kommer du att se dem i kontodetaljerna:

- Fel visar vad som gick fel
- Varje fel visar när det inträffade
- Åtgärda det underliggande problemet för att lösa fel

## Ta bort konton

**Obs**: Endast användare med Admin- eller Medarbetarroll kan ta bort konton.

För att ta bort ett konto:

1. Öppna kontot
2. Klicka på "Ta bort" eller "Radera"
3. Bekräfta borttagningen

**Obs**: Se till att inga integrationer använder kontot innan du tar bort det.
