Standout ger snabba sätt att hitta vad du behöver och röra dig mellan olika områden.

## Använda sök

Tryck på tangenten **"s"** var som helst för att öppna sökfältet.

- Sök fungerar i realtid medan du skriver
- Resultat visas automatiskt
- Klicka på ett resultat eller tryck på Enter för att öppna det
- Navigera med pil upp och pil ner
- Tryck på Escape för att stänga sökningen

Du kan söka efter:

- Organisationer efter namn
- Integrationer efter namn

## Sidomenynavigering

Använd sidomenyn för att röra dig mellan huvudområden:

- **Organisationer**: Lista över alla organisationer
- **Integrationer**: Dina arbetsflödesintegrationer
- **Konton**: Anslutningar till tjänstekonton
- **Medlemmar**: Teammedlemmar i organisationen
- **Fakturering**: Planer, prenumerationer och fakturor
- **Aktivitetslogg**: Logg över ändringar och händelser

## Sökvägar

Sökvägar visar din nuvarande plats i Standout. De visas högst upp på sidan.

- Varje sökväg är en klickbar länk
- Klicka på vilken sökväg som helst för att gå tillbaka till den sektionen
- Sökvägar hjälper dig att förstå var du är
- De gör det enkelt att navigera tillbaka till tidigare nivåer

Till exempel, om du visar en specifik integration, kan sökvägar visa: **[Organisationsnamn] > Integrationer > [Integrationsnamn]**

## Snabbnavigeringstips

- Använd sök för snabb åtkomst till specifika objekt
- Sidomenyn visar din nuvarande plats
- Klicka på organisationsnamnet för att växla organisationer
- Använd webbläsarens bakåt/framåt-knappar för att navigera i historiken
