# Kan inte logga in

Felsökningssteg om du har problem med att logga in i Standout.

## Kontrollera dina uppgifter

1. Verifiera att din e-postadress är korrekt
2. Kontrollera stavfel i ditt lösenord
3. Säkerställ att Caps Lock inte är aktiverat på ditt tangentbord
4. Försök återställa ditt lösenord

## Återställ lösenord

1. Gå till [sidan för glömt lösenord](https://app.integrationer.se/users/password/new)
2. Ange din e-postadress
3. Kontrollera din e-post för återställningsinstruktioner
4. Följ länken för att skapa ett nytt lösenord

## Har du fortfarande problem?

- Rensa webbläsarens cache
- Försök med en annan webbläsare
- Kontrollera om ditt konto är aktivt
- Kontakta support om problemen kvarstår
