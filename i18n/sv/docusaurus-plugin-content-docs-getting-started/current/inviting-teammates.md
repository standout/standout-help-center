# Bjuda in medlemmar

Lär dig hur du bjuder in medlemmar till din Standout-arbetsyta.

## Hur du bjuder in medlemmar

1. Gå till din organisation
2. Klicka på "Medlemmar"
3. Klicka på "Ny medlem"
4. Ange e-postadressen
5. Välj en roll:
   - **Admin**: Full åtkomst till alla funktioner
   - **Medarbetare**: Kan skapa och redigera integrationer
   - **Läsare**: Kan endast visa, kan inte göra ändringar
6. Klicka på "Bjud in" för att skicka inbjudan

## Hantera behörigheter

Du kan tilldela olika roller till medlemmar:

- **Admin**: Kan hantera allt inklusive fakturering, medlemmar och inställningar
- **Medarbetare**: Kan skapa och redigera integrationer och konton
- **Läsare**: Kan visa men kan inte göra ändringar
