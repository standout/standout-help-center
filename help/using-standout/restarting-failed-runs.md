# Restarting failed runs

Learn how to restart failed integration runs in Standout.

## How to restart a run

1. Go to the Integrations page
2. Find the failed run in the run history
3. Click on the failed run
4. Click "Restart run"
5. Monitor the new run status

## Troubleshooting

If a run continues to fail:
- Check the error logs
- Verify integration settings
- Test the connection manually
- Contact support if needed
