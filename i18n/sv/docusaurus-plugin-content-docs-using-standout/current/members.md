Medlemmar är de personer som har tillgång till din organisation. Du kan bjuda in medlemmar och tilldela dem olika roller.

## Visa medlemmar

Gå till fliken "Medlemmar" i en organisation för att se alla teammedlemmar.

- Listan visar alla medlemmar och deras roller
- Du kan se vem som bjöd in varje medlem
- Redan bekräftade användare får omedelbar åtkomst och ett e-postmeddelande om det.
- Obekräftade användare visas som "väntar på bekräftelse" och får en e-postinbjudan för att bekräfta sitt användarkonto och komma åt organisationen.

## Bjuda in en ny medlem

**Obs**: Endast användare med Adminroll kan bjuda in nya medlemmar.

1. Gå till Medlemmssidan
2. Klicka på "Ny medlem"
3. Ange personens e-postadress
4. Välj en roll:
   - **Admin**: Full åtkomst till alla funktioner
   - **Medarbetare**: Kan skapa och redigera integrationer
   - **Läsare**: Kan endast visa, kan inte göra ändringar
5. Klicka på "Bjud in" för att skicka inbjudan

## Medlemsroller

Varje roll har olika behörigheter:

- **Admin**: Kan hantera allt inklusive fakturering, medlemmar och inställningar
- **Medarbetare**: Kan skapa och redigera integrationer och konton
- **Läsare**: Kan visa men kan inte göra ändringar

## Redigera medlemmar

**Obs**: Endast användare med Adminroll kan redigera medlemsroller.

För att ändra en medlems roll:

1. Hitta medlemmen i listan
2. Klicka på "Redigera"
3. Välj en ny roll
4. Klicka på "Uppdatera" för att spara

_Det måste alltid finnas minst en Admin i organisationen. Men vi har validering på plats för att förhindra att du tar bort den sista Admin. Så oroa dig inte för det._

## Skicka om inbjudningar

**Obs**: Endast användare med Adminroll kan skicka om inbjudningar.

Om någon inte har bekräftat sin inbjudan:

1. Hitta medlemmen med "väntar på bekräftelse"
2. Klicka på "Skicka om"
3. Ett nytt inbjudningsmeddelande skickas

## Ta bort medlemmar

**Obs**: Endast användare med Adminroll kan ta bort medlemmar.

För att ta bort en medlem:

1. Hitta medlemmen i listan
2. Klicka på "Ta bort"
3. Bekräfta borttagningen

_Det måste alltid finnas minst en Admin i organisationen. Men vi har validering på plats för att förhindra att du tar bort den sista Admin. Så oroa dig inte för det._
