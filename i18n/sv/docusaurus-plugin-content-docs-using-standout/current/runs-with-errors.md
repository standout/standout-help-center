När en körning misslyckas visas den i listan "Körningar med fel". Detta hjälper dig att snabbt hitta och åtgärda problem.

## Hitta misslyckade körningar

Gå till sektionen "Körningar med fel" för att se misslyckade körningar.

- Listan visar alla körningar som har misslyckats
- Varje körning visar när den misslyckades och vilken integration den tillhör

## Visa felinformation

**Obs**: Alla användare med Läsare-, Medarbetare- eller Adminroll kan visa misslyckade körningar och deras detaljer.

1. Gå till sektionen "Körningar med fel"
2. Du kommer att se en lista över alla misslyckade körningar
3. Klicka på en körning för att se dess detaljer
4. Kontrollera loggarna för att förstå vad som gick fel

## Förstå felet

Varje misslyckad körning visar:

- **När den misslyckades**: Datum och tid
- **Vilken integration**: Integrationen som kördes
- **Felmeddelande**: Vad som gick fel
- **Loggar**: Detaljerad information om felet

## Åtgärda fel

**Obs**: Endast användare med Admin- eller Medarbetarroll kan åtgärda fel genom att uppdatera inställningar eller starta om körningar.

1. Öppna den misslyckade körningen
2. Läs felmeddelandet och loggarna
3. Identifiera problemet
4. Åtgärda problemet (uppdatera inställningar, kontrollera anslutningar, konfigurera beroenden i tredjepartstjänster, etc.)
5. Försök köra igen

## Förhindra framtida fel

- Kontrollera dina integrationer regelbundet
- Övervaka sektionen "Körningar med fel" för nya fel
- Åtgärda problem så snart de visas
- Håll dina kontoanslutningar uppdaterade
