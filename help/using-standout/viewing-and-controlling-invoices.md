# Viewing and controlling invoices

Learn how to view and manage invoices in Standout.

## Viewing invoices

1. Navigate to the Invoices section
2. Filter by date, status, or integration
3. Click on an invoice to view details
4. Download or print invoices as needed

## Invoice controls

- Approve or reject invoices
- Add comments or notes
- Export invoice data
- Set up automated workflows
