Integrationer (kallas också flöden och arbetsflöden) automatiserar uppgifter mellan olika tjänster. Du kan skapa, aktivera och hantera dem från sidan Integrationer.

## Skapa en ny integration

**Obs**: Endast användare med Admin- eller Medarbetarroll kan skapa integrationer.

1. Gå till organisationssidan
2. Klicka på "Ny integration" eller "Nytt arbetsflöde"
3. Välj en trigger (vad som startar integrationen)
4. Lägg till steg (vilka åtgärder som ska utföras)
5. Konfigurera inställningarna
6. Spara integrationen

## Aktiva och inaktiva flikar

Sidan Integrationer har två flikar:

- **Aktiva**: Visar integrationer som för närvarande körs
- **Inaktiva**: Visar integrationer som är pausade eller stoppade

Växla mellan flikar för att se olika integrationsstatusar.

## Integrationsstatus

Varje integration visar sin status:

- **Live**: Integrationen är aktiv och körs
- **Pausad**: Integrationen är tillfälligt stoppad
- **Utkast**: Integrationen skapas eller redigeras

## Söka efter integrationer

Använd sökfältet för att hitta specifika integrationer:

1. Skriv integrationsnamnet i sökrutan
2. Resultat filtreras medan du skriver
3. Klicka på en integration för att öppna den

## Visa integrationsdetaljer

**Obs**: Alla användare med Läsare-, Medarbetare- eller Adminroll kan visa integrationsdetaljer.

Klicka på en integration för att se:

- **Status**: Om den är live eller pausad
- **Steg**: Vad integrationen gör
- **Körningar**: Historik över när den kördes
- **Inställningar**: Konfigurationsalternativ

## Aktivera och pausa

**Obs**: Endast användare med Admin- eller Medarbetarroll kan aktivera, pausa eller ta bort integrationer.

- **Aktivera**: Klicka på "Aktivera" för att starta en integration
- **Pausa**: Klicka på "Pausa" för att tillfälligt stoppa en integration
- **Ta bort**: Ta bort en integration permanent (var försiktig med detta)

## Hantera integrationer

**Obs**: Endast användare med Admin- eller Medarbetarroll kan redigera, duplicera eller ta bort integrationer. Alla användare kan visa körningshistorik och kontrollera loggar.

Du kan:

- Redigera integrationsinställningar (endast Admin och Medarbetare)
- Visa körningshistorik (alla roller)
- Kontrollera loggar för fel (alla roller)
- Duplicera integrationer (endast Admin och Medarbetare)
- Ta bort integrationer du inte längre behöver (endast Admin och Medarbetare)
