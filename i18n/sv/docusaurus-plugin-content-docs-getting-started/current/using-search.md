Sökfunktionen hjälper dig att snabbt hitta organisationer och integrationer.

## Öppna sök

Tryck på tangenten **"s"** var som helst i applikationen för att öppna sökfältet.

Du kan också klicka på sökikonen i det övre navigeringsfältet.

## Vad du kan söka efter

Sökfunktionen hittar:

- **Organisationer**: Sök efter organisationsnamn
- **Integrationer**: Sök efter integrationsnamn

## Hur du söker

1. Tryck på **"s"** eller klicka på sökikonen
2. Skriv din sökterm
3. Resultat visas medan du skriver
4. Klicka på ett resultat för att öppna det

## Tips

- Sök fungerar i realtid medan du skriver
- Använd tydliga nyckelord för bättre resultat
- Sökningen hittar partiella matchningar i namn
